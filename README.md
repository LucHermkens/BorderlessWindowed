# BorderlessWindowed Mod

A Minecraft Fabric mod that makes Minecraft run in Borderless Windowed mode.

> **Note:** This mod and its documentation were generated by AI.

## Features

- Removes window decorations (title bar, borders) from the Minecraft window

## Installation

1. Install Fabric Loader
2. Place the mod JAR in your `mods` folder

## Building

The built JAR will be in `build/libs/`

### on Windows

```powershell
.\gradlew.bat build
```

### on Linux

```bash
./gradlew build
```

## Requirements

- Minecraft 1.21.11
- Fabric Loader >= 0.18.3
- Java 21+
